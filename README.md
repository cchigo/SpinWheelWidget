
# SpinWheelWidget

Android home screen widget built in Kotlin, wrapped as a React Native library.

## Structure

- **TappSpinDemo/** — Demo React Native app showing how to integrate the widget
- **package/** — Source code for the React Native bridge package (.tgz)
- **src/** — Kotlin Android Studio project that produces the AAR library

## How to Run the Demo
```
cd TappSpinDemo
npm install
npx react-native run-android
```

## How to Install the Library
```
npm install react-native-spin-wheel-widget-1.0.0.tgz
```
